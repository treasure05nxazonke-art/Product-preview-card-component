## ‚ö†Ô∏è Challenges & Solutions

One of the main challenges was getting the image and text layout to behave correctly on mobile while keeping the desktop design intact. Fixed heights and image sizing caused overflow and stretching issues.

I solved this by:
- Changing `flex-direction` in media queries
- Removing unnecessary height constraints on mobile
- Letting content size naturally for smaller screens
